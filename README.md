# NetJS Web Development Example

Licensed under the MIT License. See file LICENSE.

NestJS web development example. Example include JavaScript, Node.js, TypeScript.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
